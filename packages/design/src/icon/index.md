- 🔥 完全继承 antd [Icon](https://ant.design/components/button-cn) 的能力和 API，可无缝切换。
- 🆕 在 antd 图标基础上，新增上百个通用图标和 OceanBase 业务图标。

## 使用方法

使用图标组件，你需要安装 `@oceanbase/icons` 图标组件包：

```bash
npm install --save @oceanbase/icons
```
