- 🔥 完全继承 antd [Select](https://ant.design/components/select-cn/) 的能力和 API，可无缝切换。
- 💄 定制主题和样式，符合 OceanBase Design 设计规范。
- 📢 默认填充 `placeholder`，并支持通过 ConfigProvider `locale.global.placeholder` 或 `locale.Select.placeholder` 进行全局配置。
