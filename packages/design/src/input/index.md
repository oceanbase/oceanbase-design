- 🔥 完全继承 antd [Input](https://ant.design/components/input-cn) 的能力和 API，可无缝切换。
- 💄 定制主题和样式，符合 OceanBase Design 设计规范。
- 📢 默认填充 `placeholder`，并支持通过 ConfigProvider `locale.Input.placeholder` 进行全局配置。
